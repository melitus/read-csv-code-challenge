## Getting Started

The easiest way to get started is to clone the repository:

# clone the repository

```
git clone https://github.com/melitus/read-csv-code-challenge.git
```

# Change directory

```
cd server or client
```

# Install NPM dependencies

```
yarn install
```

# start the server

```
yarn run dev
```


# start the client

```
yarn start
```
